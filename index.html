本書 活用 Tips

本書をより深く活かすための4つのポイントです。

📖
手を動かしながら読む
コードは「コピペして動かす」だけで終わらせず、最初は自分でタイピングしてみましょう。書くことでコードの構造が体に馴染み、「なぜこう書くのか」という理解が自然と深まります。
🤔
「なぜそうするのか」を考える
サンプルコードや設計判断の背景には、必ず理由があります。その「なぜ」を意識して読み進めることで、応用が利く判断力が磨かれます。正解をなぞるより、理由を問う習慣を大切にしてください。
⚙️
サンプルコードをアレンジする
動いたらそれで終わりにせず、一部の値や条件を変えて試してみましょう。「うまくいく」「うまくいかない」この試行錯誤の経験こそが、実践で使える技術力の源になります。
⚖️
「作る」より「選ぶ」を意識する
カスタマイズは「できること」と「すべきこと」は違います。まず基本機能で解決できないかを考え、本当に必要なものだけを選び取る。その判断の積み重ねが、長く使われる仕組みをつくります。

よくある質問

読者からのよくあるご質問をまとめました。

最新のAPIはどこまで取り上げられていますか？
基本的には2025年12月版のkintoneアップデートまでを対象としています。一部（プラグインのパッケージング等）については、最新情報を取り入れています。JavaScript APIも新しいリリースが続いていますので、最新情報はcybozu developer network 公式ドキュメントをご確認ください。
サンプルコードはどの環境で動作確認されていますか？
2025年12月版のkintoneで動作確認しています。@kintone/rest-api-client や KUC についても、執筆時点の最新安定版を使用しています。バージョン変更による差異は Cybozu Developer Network の更新情報をご確認ください。
プログラミング未経験でも読み進められますか？
本書は kintone 認定アソシエイト取得レベルの知識があることを前提としています。kintone基本機能の知識については、kintoneヘルプ等を活用してください。kintoneカスタマイズに必要な、プログラミングについてはゼロから学べる構成です。手を動かしながら読み進めることを推奨しています。
kintone 認定カスタマイズスペシャリスト試験の対策に使えますか？
本書は kintone認定 カスタマイズスペシャリストの試験範囲をカバーしています。ただし試験合格よりも「現場で使える実践力」の習得を主眼においているため、手を動かしながら読み進めることが近道です。試験対策として活用できる練習問題をダウンロードすることができますので、知識の確認や苦手な領域の確認に活用ください。
この本を読めばプラグインが作れるようになりますか？
本書では、基本的なプラグインの構成と開発手順についての解説を行っています。実際のプラグイン開発においては、応用的な知識が必要となるため、本書の対象外としています。cybozu developer network のサンプルプラグイン等を参考にしてください。kintone認定 カスタマイズスペシャリストの試験範囲はカバーしています。
